# Cart

This is a simple application for Cart.

The entire application is contained within the `app` folder.

`main.go` is an index of the project.

You no needed to using `.env` for this one.

## Install / Sync Depencies

    go mod tidy

## Run the app

    go run main.go

## Run the tests

    go test ./... -v

## Build the app

    go build
